# 🎉 FestHub (Trial Version)

FestHub is a web-based platform designed to explore, showcase, and manage college fests and events in a centralized way.

This is a **trial deployment** of the project, hosted using **AWS S3 static website hosting**.

## 🌐 Live Demo

http://trial-festhub.s3-website.ap-south-1.amazonaws.com

## 🚀 Features

* 📅 Browse upcoming International fests & events
* 📝 Event details and information display
* 💡 Clean and responsive UI
* ⚡ Fast static hosting using AWS

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* AWS S3 (Static Website Hosting)

## 📦 Deployment

The project is deployed using AWS S3 with static website hosting enabled.

## 📌 Project Status

This is an **MVP (Minimum Viable Product)** created for learning and experimentation with frontend development and cloud deployment.
